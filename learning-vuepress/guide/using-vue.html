# primevue安裝

我們選擇的是primevue，安裝 (opens new window)。

輸入指令:
npm install primevue@^3 --save
npm install primeicons --save
示意圖

devdependencies & dependencies

DETAILS
  • 沒有dev是執行過程會用到的(dependencies)
  • plugin外掛

nuxt.config.js﹔

找到nuxt.config.js檔，加入下列程式碼﹔

    css: [
        'primevue/resources/themes/saga-blue/theme.css',
        'primevue/resources/primevue.css',
        'primeicons/primeicons.css',
        'primeflex/primeflex.css'
    ],
	build: {
		transpile: ['primevue']
	}

從plugins文件夾下建立一個primevue.js，從primevue (opens new window)網站複製primevue.js程式碼

Click me to view the code 
import { defineNuxtPlugin } from "#app";
import PrimeVue from "primevue/config";
import Button from "primevue/button";

export default defineNuxtPlugin((nuxtApp) => {
    nuxtApp.vueApp.use(PrimeVue, {ripple: true});
    nuxtApp.vueApp.component('Button', Button);
    //other components that you need
});

# ProgressBar (opens new window)元件

程式碼說明﹔

<ProgressBar :value="value" />

有「:」= 當作敘述、程式碼；無「:」= 直接設值。

全域變數﹔

放在primevue.js的button是全域變數，也就是新增其他檔案都可以使用button的功能。當我們不需要每個檔案都使用button，可以不放在primevue.js檔，在新增的檔案引入即可。

示意圖﹔

primevue.js﹔

我們將progreessbar放在about.vue檔

Click me to view the code 
<script setup>
import ProgressBar from 'primevue/progressbar';
</script>
<template>
    <h1>About</h1>
    <ProgressBar :value="90" />
</template>

# Knob (opens new window)&ProgressSpinner (opens new window)元件

Click me to view the code 
<script setup>
import ProgressBar from 'primevue/progressbar';
import Knob from 'primevue/knob';
import ProgressSpinner from 'primevue/progressspinner';
import { ref } from 'vue';
const value4 = ref(10);

</script>
<template>
    <h1>About</h1>
    <ProgressBar :value="90" />
    <Knob v-model="value4" :min="-50" :max="50" />
    <ProgressSpinner aria-label="Basic ProgressSpinner" />
</template>
