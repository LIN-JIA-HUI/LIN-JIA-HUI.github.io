# 第一堂(安裝套件&vue簡單認識)

# 環境建置

  1. git clone 檔案
  2. 下載node.js (opens new window)左邊的

  1. git switch dev
  2. npm i
  3. npm run dev
結果圖

  1. ctrl c 退出
  2. 去git bash終端機 (確認在use-me檔案，沒有的話，打cd use-me) 再打(code .)

  1. 建立自己的分支 (自己取名字)

  1. 切換到自己的分支

![](https://i.imgur.com/6BBaBjy.png =50%x)

  1. 按new folder建立檔案(取pages)

  1. 安裝vue套件﹔

# 建立檔案練習網頁架構

  1. 在pages裡面建立new file(取about.vue)

Click me to view the code 
<template>
    <h1>About</h1>
</template>

git bash指令﹔退出時按ctrl c，然後重新進入打npm run dev

網址改寫﹔http://localhost:3000/about 按住Ctrl點選第一個

在網址處點選進行更改

結果圖(TEST文字不一樣沒關係)﹔

出現下面畫面或跑不出文字，就是需要重新進入npm

  1. 新增多個檔案(在vue中預設開啟檔案會是index.vue)
  • index.vue﹔

Click me to view the code 
<template>
    <div>
        <h1>Index</h1>
    </div>
</template>

ctrl c,npm run dev 網址改寫﹔http://localhost:3000

註﹔index.vue檔，網址不用輸入index就可以跑進去，像是一個預設的網站，會直接連結到index.vue。

  • test.vue﹔ 程式碼copy網址 (opens new window)

註﹔(選項要是下圖，網站才不會出錯)﹔

每跑一個新的檔案要再ctrl c退出，重打npm run dev，更換網址 http://localhost:3000/test

結果圖﹔

  1. 建立新的folder(取名next),建完之後建立new file(取index.vue)

  • index.vue﹔
Click me to view the code 
<template>
    <div>
        <h1>Index hi </h1></div>
</template>

ctrl c,npm run dev 網頁 http://localhost:3000/next (註﹔在這裡可看到，我們新建的next folder，網址後面不用加index，就會直接跑到index.vue的內容，所以可知index檔是預設會跑出來的內容。)

  • try.vue﹔

Click me to view the code 
<template>
    <h1>index/hello</h1>
</template>

ctrl c,npm run dev 網頁 http://localhost:3000/next/try

老師貼心提醒﹔

  • 可以不用有app.vue檔。有app.vue的話就一定要放標籤，才會抓到我們的標籤(about.vue、try.vue...)。
  • scoped只對這個檔案有用

app.vue可美化一下改成如下，

  • app.vue﹔
Click me to view the code 
<script setup>
console.log('TEST')
</script>

<template>
  <div>
    <h1>TEST</h1>
    <p>
    <NuxtPage/>
    <!--<NuxtWelcome />-->
    <NuxtLink class="mybtn" to="/">Home</NuxtLink>
    <NuxtLink class="mybtn" to="/about">About</NuxtLink>
    <NuxtLink class="mybtn" to="/next">Next</NuxtLink>
    <NuxtLink class="mybtn" to="/next/try">Try</NuxtLink>
  </p>
  </div>
</template>

<style scoped>
h1 {
  background:blue;
  color:white;
  padding:12px;
}
.mybtn{
  color:blue;
  display:inline;
  padding: 12px;
  background: lightcoral;
  border: 1px solid lightgreen;
  margin-right: 16px;
  height: 48px;
}
</style>
